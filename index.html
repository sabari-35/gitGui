üß± INITIALIZING

CommandDescription
git config --global user.name "Your Name"Sets your name for all commits in Git.
git config --global user.email "Your Email"Sets your email address for commits.
git config --global init.defaultBranch mainSets main as the default branch for new repos.
git initInitializes a new Git repository in the current folder.
git add .Stages all changes (new, modified, deleted files).
git commit -m "message"Commits the staged changes with a message.
git remote add origin [URL]Connects your local repo to a GitHub repository.
git push origin mainUploads local commits to the remote main branch.
git statusShows the status of changes (staged, unstaged, untracked).
git logDisplays commit history.
git checkout [commit-hash]Moves HEAD to a specific commit.
git checkout mainSwitches back to the main branch.
git checkout -f mainForces checkout discarding any local changes.

üåø BRANCHING

CommandDescription
git branch main1Creates a new branch named main1.
git checkout main1Switches to branch main1.
git checkout -b main2Creates and switches to a new branch main2.
git branch newBranchName OldBranchNameCreates a new branch from an existing one.
git push --set-upstream origin branchNameLinks the local branch to a remote one for future pushes.
git push -u origin branchNameSame as above, sets upstream tracking branch.
git fetchDownloads updates from remote without merging them.

üîÅ PULL REQUESTS & MERGING

CommandDescription
git pull origin mainFetches and merges latest changes from remote.
git pull origin main --rebaseFetches and applies changes on top of your commits (cleaner history).
git merge branchNameMerges a branch into the current one.
git merge --abortCancels an ongoing merge safely.

üß© RESET & REVERT

CommandDescription
git resetUnstages changes but keeps them in working directory.
git reset [commit-hash]Moves HEAD to a specific commit (keeps changes unstaged).
git reset --soft [commit-hash]Moves HEAD and keeps all changes staged (for recommit).
git reset --hard [commit-hash]Completely resets to a commit, discarding all changes.
git revert [commit-hash]Creates a new commit that undoes a previous commit.

üì¶ STASHING

CommandDescription
git stashSaves your uncommitted changes temporarily.
git stash listShows a list of stashed changes.
git stash apply [stash-id]Applies a specific stash back to your working directory.
